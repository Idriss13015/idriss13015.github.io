Politique de Confidentialité

Dernière mise à jour : 20 janvier 2026

Chez Pikacheck, nous prenons votre vie privée au sérieux. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles.

1. Informations Collectées

1.1 Informations que vous nous fournissez

  • Compte utilisateur : Adresse e-mail, nom d'utilisateur
  • Données de collection : Cartes et produits ajoutés à votre collection
  • Informations de paiement : Traitées de manière sécurisée par Apple App Store ou Google Play Store (nous ne stockons pas vos informations de carte bancaire)

1.2 Informations collectées automatiquement

  • Données d'utilisation : Fonctionnalités utilisées, fréquence d'utilisation
  • Informations techniques : Type d'appareil, système d'exploitation, identifiant unique de l'appareil
  • Données analytiques : Via Firebase Analytics et Google Analytics pour améliorer l'application

2. Utilisation des Informations

Nous utilisons vos informations pour :

  • Fournir et améliorer nos services
  • Gérer votre compte et vos collections
  • Vous envoyer des notifications sur les nouvelles sorties et alertes de prix
  • Traiter vos abonnements Premium
  • Analyser l'utilisation de l'application pour améliorer l'expérience utilisateur
  • Assurer la sécurité et prévenir la fraude

3. Partage des Informations

Nous ne vendons jamais vos données personnelles. Nous pouvons partager vos informations uniquement avec :

  • Fournisseurs de services : Firebase (Google), Apple, Google Play pour le fonctionnement de l'application
  • Obligations légales : Si la loi l'exige ou pour protéger nos droits

4. Sécurité des Données

Nous utilisons des mesures de sécurité conformes aux normes de l'industrie :

  • Chiffrement des données en transit (HTTPS/SSL)
  • Authentification sécurisée via Firebase Auth
  • Stockage sécurisé dans Firebase Cloud Firestore
  • Accès limité aux données par notre équipe

5. Vos Droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous avez le droit de :

  • Accès : Demander une copie de vos données personnelles
  • Rectification : Corriger vos données inexactes
  • Suppression : Demander la suppression de votre compte et de vos données
  • Portabilité : Recevoir vos données dans un format structuré
  • Opposition : Vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : contact@pikacheck.com

6. Cookies et Technologies Similaires

Notre site web utilise :

  • Google Analytics : Pour analyser le trafic du site (anonymisé)
  • Microsoft Clarity : Pour comprendre l'utilisation du site
  • Google Tag Manager : Pour gérer les scripts de tracking

Vous pouvez désactiver les cookies dans les paramètres de votre navigateur.

7. Données des Mineurs

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans.

8. Conservation des Données

Nous conservons vos données tant que votre compte est actif. Si vous supprimez votre compte, vos données personnelles seront supprimées sous 30 jours, sauf obligation légale de conservation.

9. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de "Dernière mise à jour". Nous vous notifierons des changements importants via l'application.

10. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles :

  • Email : contact@pikacheck.com
  • Site web : https://pikacheck.com
